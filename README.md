# HR-Analytics-Attrition-Analysis
📊 HR Analytics: Employee Attrition Analysis
📌 Overview

This project analyzes employee data to identify key factors influencing attrition using Python.

🎯 Objective

To understand why employees leave the organization and identify patterns based on:

Salary
Overtime
Job Satisfaction
Experience
Department
🛠️ Tools & Technologies
Python
NumPy
Pandas
Matplotlib
Seaborn
📂 Dataset
IBM HR Analytics Dataset
Contains employee details such as age, income, department, job satisfaction, and attrition status
🔧 Steps Performed
1. Data Cleaning
Handled missing values
Removed duplicates
Converted data types
Standardized categorical values
2. Exploratory Data Analysis (EDA)
Analyzed distribution of employees
Compared attrition across departments
Studied salary and satisfaction impact
3. Visualization
Attrition count
Department vs attrition
Overtime vs attrition
Salary distribution
Correlation heatmap
🔍 Key Insights
Employees working overtime are more likely to leave
Low salary employees show higher attrition
Sales department has the highest attrition
Job satisfaction significantly impacts retention
Younger employees tend to leave more frequently
💡 Recommendations
Reduce excessive overtime
Improve salary structure for low-income employees
Focus on employee satisfaction programs
Provide career growth opportunities
